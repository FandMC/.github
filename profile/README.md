# FandMC

[简体中文](./README_CN.md) | English

FandMC builds **Fand**, a modern Minecraft server platform focused on clean plugin APIs, high-performance server internals, and a practical development workflow for Java plugin authors.

## Projects

- **Fand**: the main server implementation.
- **FandPluginGradle**: Gradle tooling for creating and building Fand plugins.
- **Fandclip**: launcher and distribution tooling for runnable Fand server builds.
- **FandDocs**: official documentation for server owners and plugin developers.

## Getting Started

Documentation:

https://fandmc.cn

Maven repository:

```kotlin
repositories {
    maven("https://repo.fandmc.cn/repository/maven-public/")
}
```
