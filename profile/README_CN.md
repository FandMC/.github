# FandMC

简体中文 | [English](./README.md)

FandMC 是围绕 **Fand** 构建的 Minecraft 服务端生态。

Fand 是一个现代化插件服务端，专注于清晰的插件 API、高性能服务端实现，以及更适合现代 Java 开发体验的插件开发工具链。

## 项目

- **Fand**：主服务端项目。
- **FandPluginGradle**：用于创建和构建 Fand 插件的 Gradle 工具。
- **Fandclip**：Fand 服务端启动与分发工具。
- **FandDocs**：面向服主和插件开发者的官方文档。

## 开始使用

官方文档：

https://fandmc.cn

Maven 仓库：

```kotlin
repositories {
    maven("https://repo.fandmc.cn/repository/maven-public/")
}
```
